Css

Basic Css Structure

Styling HTML elements requires 3 things in CSS.

  • Selector
  • Property
  • Value

Selectors

Element Selector

The element selector selects elements based on the element name.

ID Selector

The id selector uses the id attribute of an element to select a element.
To select an element with a specific id, write # followed by the id of the element.

Class Selector

The class selector selects elements with a class attribute.
To select elements with a class, write . followed by the class name.

Universal Selector

The universal selector * selects all elements on the page.

Grouping Selector

The grouping selector selects all the elements with the same style definitions.
You have to write a , in between of the elements to group-select them.

Combinators

Descendant Selector

The descendant selector matches all elements that are descendants of a specified element.

For example:
div p {
 color: yellow;
}

Child Selector (>)

The child selector selects all elements that are the children of a specified element.

For example:
div > p {
 background-color: yellow;
}

Adjacent Sibling Selector (+)

The adjacent sibling selector is used to select an element that is directly after another specific element.

For example:
div + p {
 background-color: yellow;
}

General Sibling Selector (~)

The general sibling selector selects all elements that are next siblings of a specified element.

For example:
div ~ p {
 background-color: yellow;
}

Pseudo Classes

  • :link
  • :visited
  • :hover
  • :active

Box Model

Margin is the space between the border and the surrounding elements.
Padding is the space between the content of the element and its border.
Block elements can have margin and padding to keep distance from all surrounding elements.
Inline elements stay in the line with their neighboring elements, while block elements don't.

To make an image float on one side of the website, you can use the float property:

image {
 float: right;
}
