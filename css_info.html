Basic Css Structure

Styling HTML elements requires 3 things in CSS.

  • Selector
  • Property
  • Value

Selectors

Element Selector

The element selector selects elements based on the element name.

ID Selector

The id selector uses the id attribute of an element to select a element.
To select an element with a specific id, write # followed by the id of the element.

Class Selector

The class selector selects elements with a class attribute.
To select elements with a class, write . followed by the class name.

Universal Selector

The universal selector * selects all elements on the page.

Grouping Selector

The grouping selector selects all the elements with the same style definitions.
You have to write a , in between of the elements to group-select them.

Combinators

Descendant Selector

The descendant selector matches all elements that are descendants of a specified element.

For example:
div p {
 color: yellow;
}

Child Selector (>)

The child selector selects all elements that are the children of a specified element.

For example:
div > p {
 background-color: yellow;
}

Adjacent Sibling Selector (+)

The adjacent sibling selector is used to select an element that is directly after another specific element.

For example:
div + p {
 background-color: yellow;
}

General Sibling Selector (~)

The general sibling selector selects all elements that are next siblings of a specified element.

For example:
div ~ p {
 background-color: yellow;
}

Pseudo Classes

A CSS pseudo-class is a keyword added to a selector that specifies a special state of the selected element, e.g.:

  • :link
  • :visited
  • :hover
  • :active

Lists

  • Make bullet a square (list-style-type: square)
  • Make bullet an image named "my-bullet.png" (list-style-image: url("my-bullet.png"))
  • Make bullet an uppercase roman number (list-style-type: upper-roman)
  • Set bullet position to be inside the list item's text flow (list-style-position: inside)
  • Set bullet position to be outside the list item's text flow (list-style-position: outside)

Gradients

CSS gradients let you display smooth transitions between two or more specified colors.
CSS defines three types of gradients:

  • Linear Gradients (goes down/up/left/right/diagonally)
  • Radial Gradients (defined by their center)
  • Conic Gradients (rotated around a center point)

Box Model

Margin is the space between the border and the surrounding elements.
Padding is the space between the content of the element and its border.
Block elements can have margin and padding to keep distance from all surrounding elements.
Inline elements stay in the line with their neighboring elements, while block elements don't.

To make an image float on one side of the website, you can use the float property:
image {
 float: right;
}

To stop a section element from flowing around a right floating image, you can use the clear property:
section {
 clear: right;
}

Flexbox

The flexible box layout module, usually referred to as flexbox, was designed as a one-dimensional layout model, and as a method that could offer space distribution between items in an interface and powerful alignment capabilities.
When we describe flexbox as being one-dimensional we are describing the fact that flexbox deals with layout in one dimension at a time â€” either as a row or as a column. This can be contrasted with the two-dimensional model of CSS Grid Layout, which controls columns and rows together.

Animations

An animation lets an element gradually change from one style to another.
You can change as many CSS properties you want, as many times as you want.
To use CSS animation, you must first specify some keyframes for the animation.
Keyframes hold what styles the element will have at certain times.

For example:

@keyframes moveSquare {
 25% {
  background-color: yellow;
  transform: translate(10rem, 0rem) rotate(-90deg) scale(150%, 150%);
 }
 50% {
  background-color: green;
  transform: translate(10rem, 10rem) rotate(-180deg) scale(200%, 200%);
 }
 75% {
  background-color: blue;
  transform: translate(0rem, 10rem) rotate(-270deg) scale(150%, 150%);
 }
 100% {
  background-color: red;
  transform: translate(0rem, 0rem) rotate(-360deg) scale(100%, 100%);
 }
}
