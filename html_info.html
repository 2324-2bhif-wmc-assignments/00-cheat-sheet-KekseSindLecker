HTML

Basic structure

<!DOCTYPE html>
<html lang="en">
 <head>
  <meta charset="UTF-8">
  <title>Title</title>
 </head>
 <body>
 </body>
</html>

Non-Void Elements

Non-Void Elements consist of three things:

  • <Opening Tag>
  • Content
  • </Closing Tag>

Void Elements

Void elements in HTML are self-closing tags like <img>, <br>, and <input> that don't have a closing tag and don't contain content.
Example: <link type="text/css" rel="stylesheet" href="style.css">

Attributes

Attributes provide additional information about elements.
Attributes are e.g. "class", "src", "href".

Some Elements/Tags

Anchor Tag

The <a> element in HTML is used to create hyperlinks, allowing you to link to other web pages or resources on the internet like.

Article Tag

The <article> tag specifies independent, self-contained content.

Section Tag

The <section> element is a structural HTML element used to group together related elements.

Image Tag

The <img> tag is used for placing a picture on a website.

Paragraph Tag

The <p> HTML element represents a paragraph.

Span Tag

You use the <span> element to wrap sections of text for styling purposes.

Linking A Stylesheet

To style your website, you have CSS.
To include the CSS to your HTML file, write the following into the head:
<link type="text/css" rel="stylesheet" href="{CSS filename}">
