Cheat Sheet

World Wide Web General

  • HTTP (Hypertext Transfer Protocol) is the foundation of data communication on the web. It's a protocol that defines how information is exchanged between a web and a web server.
  • HTML (Hypertext Markup Language) is the standard language used to create and structure content on the World Wide Web.
  • A URL (Uniform Resource Locator) is a reference to a resource that specifies its location on the internet.
    The syntax of a URL is as follows:
    scheme ":" ["//" authority] path ["?" query] ["#" fragment]

Web Server

  • A web server is computer software and underlying hardware that accepts requests via HTTP(S).
  • It accepts (or refuses) requests from web clients and serves them web pages in response.
  • Web servers often store files in their file system. These files can include HTML documents, images, videos, scripts, and more.
  • Web content refers to the textual, aural, or visual content published on a website.

Web Client

  • A web client is a client-side application used for connecting to a web server over HTTP(S).
  • It interacts with DNS/web servers to access and retrieve information from the World Wide Web.
  • Web clients and web browsers are not the same, but they often interact with each other.

Hello World

  • <!DOCTYPE html> is used to declare that the document is written in HTML
  • The <head> element contains meta-information about the document that is not displayed on the webpage itself.
  • The two top most elements within the html element are <head> and <body>.
  • Emphasizing/Marking up text in web content can be done using various HTML tags and CSS styles.

HTML

Basic structure

<!DOCTYPE html>
<html>
 <head>
  <meta charset="UTF-8">
  <title>Title</title>
 </head>
 <body>
 </body>
</html>

Void Elements and Attributes

Void elements in HTML are self-closing tags like <img>, <br>, and <input> that don't have a closing tag and don't contain content.
Example: <link type="text/css" rel="stylesheet" href="style.css">
Attributes are e.g. "type", "rel", "href".

Anchor Tag

The <a> element in HTML is used to create hyperlinks, allowing you to link to other web pages or resources on the internet like:

<a href="https://youtube.com">
 youtube.com
</a>

Css

Basic Css Styling

Example:

h1 {
 font-family: Arial;
 color: darkblue;
}

Classes

In HTML, you can create classes by doing this:
<p class="my-class">This is a paragraph with a class.</p>

To use this class with Css, you have to set the selector to the .{class-name}.

Example:

.my-class {
 font-size: large;
 background-color: red;
}
