What is Bootstrap?

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development.
It contains HTML, CSS and JavaScript-based design templates for typography, forms, buttons, navigation, and other interface components.

Pre-styled List

To easily create a list that doesn't look like it's your first time trying CSS,
use the "list-group" and "list-group-item" classes:
<ul class="list-group">
 <li class="list-group-item">Lorem ipsum</li>
 <li class="list-group-item">dolor</li>
 <li class="list-group-item">sit amet</li>
</ul>

  • Lorem ipsum
  • dolor
  • sit amet

Pre-styled Table

To easily create a table that doesn't look like it's your first time trying CSS, use the "table" class:
<table class="table">
 <thead>
  <tr>
   <th>Firstname</th>
   <th>Lastname</th>
   <th>Age</th>
  </tr>
 </thead>
 <tbody>
  <tr>
   <td>Jill</td>
   <td>Smith</td>
   <td>50</td>
  </tr>
  <tr>
   <td>Eve</td>
   <td>Jackson</td>
   <td>24</td>
  </tr>
  <tr>
   <td>John</td>
   <td>Doe</td>
   <td>34</td>
  </tr>
 </tbody>
</table>

Card

To create a card, you can use the following code snippet:
<div class="card" style="width: 20rem;">
 <div class="card-body">
  <h4 class="card-title">Card title</h4>
  <p class="card-text">Some quick example text to build on the card title and make up the bulk of the card's content.</p>
  <a href="#" class="btn btn-primary">Go somewhere</a>
 </div>
</div>
